Cicerone lets you parse OpenAPI/Swagger schemas into Python objects for inspection, traversal, and tooling.

## Installation

Install cicerone as a global CLI tool:

### With pip (Python)

```sh
pipx install cicerone
```

### With uv (Python)

```sh
uv add cicerone
```
